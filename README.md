# 🛡️ Toolkit Rischio 360°

Sistema multiaziendale di mappatura e gestione dei rischi aziendali.

## 🚀 Come usare

L'app è disponibile online su:
**https://massimocoletta1959-wq.github.io/toolkit-rischio-360**

## ✨ Funzionalità

- **Registro Rischi** — Identifica, categorizza e valuta i rischi con scala P×I
- **Piano d'Azione** — Azioni di mitigazione collegate ai rischi, con responsabile e scadenza
- **Cruscotto** — Heatmap, statistiche e avanzamento aggiornati in tempo reale
- **Multi-azienda** — Ogni account vede solo i dati della propria azienda

## 🛠️ Sviluppo locale

```bash
npm install
npm start
```

## 📦 Deploy

Il deploy è automatico via GitHub Actions su ogni push al branch `main`.
